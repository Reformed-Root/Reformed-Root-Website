About Reformed-Root

Reformed-Root is a name that carries both weight and hope.
In tech, the "root" user represents full control.
In life, it's a symbol of starting over from the ground up.
This site is built by someone who's done just that —
someone who’s walked through hard times, faced the consequences, and is now choosing to build instead of break.
“Reformed” doesn’t just mean changed — it means reimagined, restructured, and reborn.

This is a personal Flask web application built to practice and demonstrate my skills. It includes projects like a CTF-style challenge, serving as both a portfolio and a learning exercise.
